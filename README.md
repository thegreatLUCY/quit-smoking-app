# QuitTrack

QuitTrack is a simple progressive web app for tracking a quit-smoking journey with milestones, achievements, and health improvements.

## Files

- `index.html` - main app page
- `styles.css` - app styling
- `app.js` - main app behavior
- `i18n.js` - language text
- `milestones.js` - milestone data
- `manifest.json` - PWA settings
- `sw.js` - service worker
- `icons/` - PWA icons

## Deployment

This is a static app. On Netlify, use:

- Build command: leave empty
- Publish directory: `.`
